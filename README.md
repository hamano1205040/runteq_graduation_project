## サービス概要
日々のトレーニング内容を自由に記録・共有できる筋トレ記録アプリです。
自分のトレーニング履歴を可視化し、継続をサポートします。
「楽しく続けること」を目的に、トレーニングの習慣化を後押しするサービスです。

## このサービスへの思い・作りたい理由
私は10年以上トレーニングを継続してきました。
30歳を過ぎた頃から、周囲では肩こりや怪我など、身体の不調に悩む人が増えてきたと感じています。
実際、デスクワーク中心の仕事をしている妻も肩こりに悩まされています。

一方で、私のトレーニング仲間を見てみると、肩こりや大きな怪我に悩まされている人はほとんどいません。
この経験から、トレーニングは見た目や競技目的だけでなく、健康維持のために誰しもが取り組むべきものだと強く感じるようになりました。

しかし、トレーニングを継続することは簡単ではありません。
ジムでは毎年、夏になると人が増え、夏が終わる頃には姿を見なくなる人を何度も見てきました。

そこで、トレーニングを「義務」ではなく「楽しく続けられるもの」にしたいと考え、このサービスを作ろうと思いました。
ジム代は医療保険のようなものだと思っています。
多くの人が健康でい続けるきっかけになるサービスを目指しています。

## ユーザー層について
- トレーニングをすでに習慣化している人
- これからトレーニングを始めたい・頑張りたいと考えている人
- 現在はトレーニングをしていないが、健康維持に関心がある人

特に、これからトレーニングを継続していきたい人をメインターゲットとしています。

## サービスの利用イメージ
ユーザーは、その日に行ったトレーニング内容を自由に記録します。
記録を続けることで自分のトレーニング履歴が蓄積され、達成感を得られます。

また、他のユーザーのトレーニング内容を見ることで刺激を受け、
「今日も少しだけでもやってみよう」と思えるきっかけを提供します。

本サービスを通して、トレーニングを楽しく継続し、
最終的にトレーニングを生活の一部として習慣化してもらうことを目指しています。

## ユーザーの獲得について
まずはRUNTEQのコミュニティを中心にユーザーを獲得する予定です。
68期のメンバーとはオフラインでの交流も多く、実際に使ってもらいながらフィードバックを得たいと考えています。

その後は、トレーニングや健康に関心のある層に向けて、SNSなどを活用した発信も検討しています。

## サービスの差別化ポイント・推しポイント
トレーニング記録アプリ自体は多く存在しますが、本サービスでは以下の点で差別化を図ります。

- 他ユーザーのトレーニング記録に対してコメントやいいねを送れる仕組み（構想段階）
- トレーニング部位ごとに成長を可視化する五角形パラメーター
- 鍛えられていない部位を「弱点」として分かりやすく提示する仕組み

単なる記録にとどまらず、継続するモチベーションを生み出すことを重視しています。

## 機能候補

### MVPリリース時に実装予定の機能
- ユーザー登録・ログイン機能
- トレーニング記録のCRUD機能
  - 日付
  - トレーニング内容（自由入力）
- 自分のトレーニング履歴一覧表示

### 本リリースまでに実装したい機能
- 他ユーザーのトレーニング記録閲覧機能
- トレーニング記録へのコメント・いいね機能
- 部位別トレーニング量を可視化する五角形パラメーター
- 弱点部位に対するフィードバック表示
