## サービス概要

トレーニングログは、日々のトレーニング内容を自由に記録・共有できる筋トレ記録アプリです。

自分のトレーニング履歴を可視化することで、トレーニングの継続をサポートします。
「楽しく続けること」を目的に、トレーニングを習慣化するきっかけを提供するサービスです。

---

## このサービスへの思い・作りたい理由

私は10年以上トレーニングを継続してきました。

30歳を過ぎた頃から、周囲では肩こりや怪我など、身体の不調に悩む人が増えてきたと感じています。
実際、デスクワーク中心の仕事をしている妻も肩こりに悩まされています。

一方で、私のトレーニング仲間を見てみると、肩こりや大きな怪我に悩まされている人はほとんどいません。

この経験から、トレーニングは見た目や競技目的だけでなく、健康維持のために誰しもが取り組むべきものだと強く感じるようになりました。

しかし、トレーニングを継続することは簡単ではありません。
ジムでは毎年、夏になると人が増え、夏が終わる頃には姿を見なくなる人を何度も見てきました。

そこで、トレーニングを「義務」ではなく「楽しく続けられるもの」にしたいと考え、このサービスを作りました。

ジム代は医療保険のようなものだと思っています。
多くの人が健康でい続けるきっかけになるサービスを目指しています。

## ユーザー層について

- トレーニングをすでに習慣化している人
- これからトレーニングを始めたい・頑張りたいと考えている人
- 現在はトレーニングをしていないが、健康維持に関心がある人

特に、これからトレーニングを継続していきたい人をメインターゲットとしています。

---

## サービスの利用イメージ

ユーザーは、その日に行ったトレーニング内容を自由に記録します。

記録を続けることで、自分のトレーニング履歴が蓄積され、達成感を得られます。

また、他のユーザーのトレーニング内容を見ることで刺激を受け、
「今日も少しだけでもやってみよう」と思えるきっかけを提供します。

本サービスを通して、トレーニングを楽しく継続し、
最終的にトレーニングを生活の一部として習慣化してもらうことを目指しています。

## サービスの差別化ポイント・推しポイント

### なぜこのアプリが必要なのか

トレーニングの成果は、1日や2日で目に見えて現れるものではありません。
筋肉がついて見た目が変わるまでには、数ヶ月、場合によっては年単位の継続が必要です。

特にトレーニング初心者の場合、

- 今日もトレーニングはしたけれど、本当に意味があるのか分からない
- 成長している実感が持てず、モチベーションが下がってしまう

といった状態に陥りやすいと感じています。

このアプリでは、
「成長しているか分からない不安」
「成果が見えないことによる挫折」
を解消することを大きな目的としています。

### 五角形パラメーターによる成長の可視化

本サービスの最大の特徴は、トレーニングの成果を五角形パラメーターとして可視化する点です。

日々のトレーニング記録を、以下の5部位に分けて管理します。

- 胸
- 背中
- 脚
- 肩
- 腕

各部位のトレーニング内容を記録していくことで、それぞれの部位に対応した数値が蓄積され、五角形のレーダーチャートとして表示されます。

トレーニングを継続するほど、五角形は少しずつ大きくなっていきます。

これにより、

- 自分がどの部位をよく鍛えているのか
- 鍛えられていない部位はどこか
- トレーニングのバランスが取れているか

を視覚的に把握できるようになります。

### 他のトレーニングアプリとの違い

トレーニング記録アプリは数多く存在しますが、本サービスでは単なる記録にとどまりません。

- 記録すること自体がモチベーションにつながる設計
- 日々の積み重ねが成長として実感できる仕組み
- 初心者でも迷わず続けられる設計

を重視しています。

短期間で結果が出にくいトレーニングだからこそ、
「続けられる仕組み」と「成長を実感できる仕組み」を提供します。

## 機能候補

### MVPリリース時に実装予定の機能

- ユーザー登録・ログイン機能
- トレーニング記録のCRUD機能
  - 日付
  - トレーニング内容（自由入力）
- 自分のトレーニング履歴一覧表示

### 本リリースまでに実装したい機能

- 他ユーザーのトレーニング記録閲覧機能
- トレーニング記録へのコメント・いいね機能
- 部位別トレーニング量を可視化する五角形パラメーター
- 弱点部位に対するフィードバック表示
