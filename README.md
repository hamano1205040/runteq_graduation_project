# トレーニングログ

日々のトレーニングを記録・可視化し、楽しく継続できる筋トレ記録アプリ

---

## 📋 サービス概要

**トレーニングログ**は、日々のトレーニング内容を自由に記録・共有できる筋トレ記録アプリです。

- トレーニング履歴を可視化し、継続をサポート
- 「楽しく続けること」を目的に、習慣化を後押し
- 初心者でも目に見える形で成長を実感できる仕組み

---

## 💡 サービスへの思い

このアプリは、自身のトレーニング経験と周囲の健康問題から着想を得ました。

トレーニングは義務ではなく、**楽しく続けられるもの**であってほしい。そんな思いから、このサービスを作りました。

---

## 🎯 想定ユーザー層

- すでにトレーニングを習慣化している人
- これからトレーニングを始めたい・頑張りたい人
- 健康維持に関心がある人

---

## 🔄 サービスの利用イメージ

1. **トレーニング記録の入力** - 日付と内容を自由に記録
2. **履歴確認** - 自分の過去のトレーニングを振り返る
3. **他ユーザーの記録閲覧** - 他の人の頑張りを見てモチベーションアップ
4. **継続のきっかけ** - 可視化された成長を見て、次のトレーニングへ

---

## 🚀 ユーザー獲得方法

- **RUNTEQコミュニティ**でのフィードバック収集
- **SNS**などを活用した情報発信

## ⭐ 差別化ポイント（推しポイント）

### なぜこのアプリが必要なのか？

トレーニングの成果は、**1日2日では出ません**。筋肉がついて見た目が変わるのは、数ヶ月、長ければ年単位の積み重ねです。だからこそ、**日々のトレーニングで達成感ややりがいを感じにくい**という課題があります。

特に初心者にとっては、「今日も頑張ったけど、これで本当に成長してるのかな？」という不安がつきもの。

### 🔥 五角形パラメーターで成長を"見える化"

このアプリの最大の特徴は、**五角形パラメーター**です。

- **5つの部位（胸・背中・脚・肩・腕）ごとのトレーニング量を、レーダーチャートで可視化**
- トレーニングを記録するたびに、五角形が少しずつ大きくなっていく
- 「今週は胸ばかり鍛えてたから、背中を重点的にやろう」といった**バランス調整の目安**になる
- 初心者でも「今日はどの部位を鍛えればいいのか」が一目でわかる

### ✅ 他のアプリとの違い

多くのトレーニングアプリは「記録」に特化していますが、このアプリは違います。

- **記録すること自体がモチベーションになる**設計
- 日々の積み重ねが視覚的にわかるから、**「続けたい」と思える**
- 五角形が大きくなっていくのを見るだけで、達成感を感じられる
- 初心者でも迷わず、**習慣化・継続をサポート**する仕組み

### 💬 コミュニティ機能（構想段階）

他のユーザーのトレーニング記録に**コメントやいいね**を送れる仕組みを検討中。仲間と励まし合いながら、楽しく続けられる環境を作ります。

**「今日も1歩、成長した」** ── その実感が、明日のトレーニングを支えます。
